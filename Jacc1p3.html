JACC 1.3

- Just Another Compiler Compiler

- a YACC-like application

- a parser generator written in Java which generates code implementing the LR parsing algorithm driven by an SLR(1) parsing table constructed from a BNF specification (parse that!)

- built around the ideas of chapter 4 of The Dragon Book (LR parsing algorithm and SLR construction) and parts of Lex & Yacc by Levine, et al (YACC input format)


Features beyond JACC 1.2:

- there is now an option to generate a Python implementation for the parser
